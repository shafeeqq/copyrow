How to Use:
1 : Download copy.plugin.js and include javascripts 

<script type="text/javascript" src="http://code.jquery.com/jquery-1.10.2.min.js"></script>

<script type="text/javascript" src="copy.plugin.js"></script>
2 : There should be one button for copy and put class name for the row you need to copy,also put parent class name for the row. 

    <div class="parent">
     <div class="row">
       <div>Name</div>  <input type="text" name="sample_test" id="sample_test" class="txtbox" value="">
        <input type="button" value="+" class="copy">
      </div>
    </div>
3 : Pass the Configuration options inside the event of copy button 

   $(".copy").on("click",function(){
    
                $(this).copyrow({
                "maximum" :"5",
                "copyfrom" : "row",
                "custommessage" : "Maxixmum record exceed",
                "dynamicname" : true,
                "removeclass" : "deleted",
                "parentclass"    : "parent"
               });
  
    });
